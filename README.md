斗兽棋，亦称动物棋，是中国传统棋类游戏之一，双方各执八种动物棋子，在特定棋盘上进行对弈。‌

‌基本构成与规则‌
棋盘通常为横九竖七的方格布局，共六十三格，其中设有两条规格为2*3的“河流”区域。棋子分为八种，按传统等级排序为：象、狮、虎、豹、狼、狗、猫、鼠。双方各持一套，共十六枚棋子。‌

核心规则遵循“大吃小”原则，即等级高的动物可吃等级低的动物。但存在两项特殊规则：一是鼠可以吃象；二是同类动物相遇可互相吃掉。此外，狮子和老虎拥有“跳河”能力，可纵横对直跳过河流，若河中有老鼠阻隔则无法跳跃。老鼠可走入河中，在河内时，陆地动物无法攻击它，河中的老鼠也不能吃陆地上的象；若两鼠在河中相遇，则可互吃。‌

棋盘上每方兽穴旁设有三个“陷阱”。敌方动物走入陷阱即失去战斗力，己方任何动物均可将其吃掉。己方动物不受陷阱限制。‌

‌胜负判定‌
主要胜利条件有两种：一是己方任何棋子成功走入敌方兽穴；二是将对方所有棋子全部吃掉。‌
